# Raytracing-engine

Simple software renderer using [Raytracing method](https://en.wikipedia.org/wiki/Ray_tracing_(graphics))

# Get Started

You can simply compile it using CMake
```console
$ mkdir build
$ cd build
$ cmake ..
$ make
```
